## Specpin làm gì

Specpin gắn các đặc tả nghiệp vụ (quy tắc, mô tả, tiêu chí nghiệm thu) trực tiếp
lên các phần tử của một giao diện web *đang chạy*, rồi hiển thị ngay trong trình
duyệt khi bạn rê chuột hoặc duyệt trang. Specpin **không** sinh code: nó ghim tài
liệu luôn cập nhật, được quản lý phiên bản bằng Git, lên chính giao diện bạn đang có.

## Cách hoạt động

<div class="specpin-flow" not-content>
  <div class="step">
    <span class="n">1</span>
    <h3>Đặc tả nằm trong repo</h3>
    <p>Đặc tả là JSON trong thư mục <code>.specs/</code> của repo: quản lý phiên
    bản, duyệt qua PR, và so sánh diff được.</p>
  </div>
  <div class="step">
    <span class="n">2</span>
    <h3>Sidecar phục vụ chúng</h3>
    <p><code>specpin serve</code> mở <code>.specs/</code> qua một API localhost
    có xác thực token kèm live-reload.</p>
  </div>
  <div class="step">
    <span class="n">3</span>
    <h3>Extension hiển thị chúng</h3>
    <p>Extension trình duyệt khớp mỗi đặc tả với một phần tử thực trên trang và hiển thị nó
    ngay trên trang.</p>
  </div>
</div>

```text
.specs/ (trong repo)  -->  specpin serve (localhost HTTP + SSE)  -->  extension trình duyệt (khớp + hiển thị)
```

## Bắt đầu nhanh

```bash
# 1. Cài CLI
npm install -g @specpin/cli    # hoặc: pnpm add -g @specpin/cli

# 2. Trong repo dự án của bạn: khởi tạo và phục vụ đặc tả
specpin init                   # tạo .specs/manifest.json
specpin serve                  # in ra URL localhost + bearer token

# 3. Cài extension và kết nối
#    Chrome:  Chrome Web Store (link bên dưới)
#    Firefox: Firefox Add-ons (link bên dưới)
```

Cài Specpin từ [Chrome Web Store](https://chromewebstore.google.com/detail/specpin/kkfmoieoahdjneagognaoedggkiiolkn) hoặc [Firefox Add-ons](https://addons.mozilla.org/vi/firefox/addon/specpin/), dán URL và token vừa in vào extension, mở ứng dụng của bạn, và đặc tả sẽ hiển
thị trên đúng phần tử của nó.
