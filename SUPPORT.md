# Support

Thanks for using Specpin. Here is where to go, by intent:

## Questions, usage, ideas

Ask in [GitHub Discussions](https://github.com/lamngockhuong/specpin/discussions). This is the best place for "how do I...", setup help, and feature ideas before they become issues.

## Documentation

- End-user site: <https://specpin.ohnice.app>
- Run guide (init -> serve -> load -> connect -> render -> capture): [`docs/run-guide.md`](./docs/run-guide.md)
- Schema reference: [`docs/schema-reference.md`](./docs/schema-reference.md)

Please read the run guide before filing a bug.

## Bugs and feature requests

Open an [issue](https://github.com/lamngockhuong/specpin/issues/new/choose) using the provided templates. Include your browser, extension version, and OS for bugs.

## Security

Do not open a public issue for vulnerabilities. Report them privately via a [security advisory](https://github.com/lamngockhuong/specpin/security/advisories/new). See [`.github/SECURITY.md`](./.github/SECURITY.md).
